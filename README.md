# Scholaro Homepage Redesign Mockup

A static concept homepage redesign mockup for www.scholaro.com focused on:
- clearer visual hierarchy
- reduced header clutter
- stronger CTA structure
- better trust signaling
- more product-led navigation
- improved whitespace and consistency
- cleaner utility/report-check treatment

This is an unofficial conceptual redesign for critique/demo purposes.
